Running the Project on Google Colab
1. Download the Notebook
Click on the notebook file (.ipynb) in this repository.
Press the Download button or select Download raw file.

2. Open Google Colab
Go to Google Colab

3. Upload the Notebook
Click File → Upload notebook
Select the downloaded .ipynb file from your computer.

4. Run the Project
Click Runtime → Run all
The notebook will execute all cells and generate outputs, analysis, and visualisations.
